# 🖼️🔐 PicSecret 图密

> **把秘密藏在图片里** — 纯前端 PNG 隐写通信工具

无需服务器 · 浏览器直开 · 中英双语 · AES-256 加密

**在线使用：** [https://ydsgangge-ux.github.io/picsecret/](https://ydsgangge-ux.github.io/picsecret/)

---

## ✨ 怎么用

### 🔒 编码（隐藏信息）
1. 上传一张 **PNG 图片**
2. 输入你想隐藏的秘密信息
3. 可选：设置加密口令
4. 点击编码 → 下载图片 → 发给对方

### 🔓 解码（查看信息）
1. 上传加密图片
2. 输入口令（如果有）
3. 自动显示隐藏内容

---

## 🔧 一句话原理

信息藏在 PNG 文件的元数据区域，**图片看起来完全正常**，只有用本工具才能读取隐藏内容。

可选 AES-256 加密，不知道口令的人即使拿到图片也解不开。
